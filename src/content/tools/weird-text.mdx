# Weird Text Generator Online

Create bizarre, unusual weird text with rare Unicode characters and effects. Perfect for unique bios, strange Discord profiles, and standout social posts. Free.

## How This Tool Works

This text transformation runs entirely in your browser using JavaScript. It maps standard characters to visually distinct Unicode code points or applies computational transformations. No data is sent to any server.

The output is standard Unicode text that can be copied and pasted into any platform that supports Unicode.

## Example

**Input:** `Hello`  
**Output:** `H̷e̷l̷l̷o̷`

## Unicode Blocks Used

This tool uses **Combining Diacritical Marks** (Unicode range U+0300–U+036F). These are zero-width characters that stack above, below, or through a base character. By layering multiple combining marks per letter, the text appears distorted or "glitched." This technique is sometimes called "Zalgo text" after an internet meme.
