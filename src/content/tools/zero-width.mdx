# Zero Width Character Generator

The **Zero Width Character Generator** creates invisible Unicode characters that occupy zero visual space but are treated as real text by apps and platforms. These characters ‚Äî including Zero-Width Space (ZWSP), Zero-Width Joiner (ZWJ), and Zero-Width Non-Joiner (ZWNJ) ‚Äî are essential for creating blank names, hidden text watermarks, and bypassing "empty field" restrictions on social media.

## Types of Zero-Width Characters

| Character                 | Unicode  | Use Case                                   |
| :------------------------ | :------- | :----------------------------------------- |
| **Zero-Width Space**      | `U+200B` | Word boundaries without visible space      |
| **Zero-Width Joiner**     | `U+200D` | Joins characters (used in emoji sequences) |
| **Zero-Width Non-Joiner** | `U+200C` | Prevents ligature formation                |
