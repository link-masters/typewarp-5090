# Box Text Generator Online

Wrap your text in decorative Unicode box frames and borders. Perfect for Discord announcements, social media bios, and creative headers. Free, instant results.

## How Box Text Works

This tool wraps each character in a box using Unicode **Enclosed Alphanumerics** (U+2460–U+24FF). These characters display each letter or digit inside a square or rounded box. Originally designed for numbered lists and UI elements, they create a structured, emphasis-heavy appearance when used for text.

For multi-line box frames, the tool can also use **Box Drawing** characters (U+2500–U+257F) to create custom borders around blocks of text.

**Unicode source:** Enclosed Alphanumerics block and Box Drawing block. All processing runs locally in your browser.

## Example

**Input:** `HELLO`  
**Output:** `🄷🄴🄻🄻🄾`
