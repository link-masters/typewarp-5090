# Slash Text Generator

Create edgy slashed text with diagonal Unicode combining lines. Perfect for technical branding, unique headers, and standout social media bios. Free online.

## How Slash Text Works

This tool applies the **Combining Long Solidus Overlay** (U+0338) to each character. Unlike horizontal strikethrough (U+0336), the solidus overlay draws a diagonal slash through the character, creating a more aggressive crossed-out effect commonly used in technical documentation, mathematical notation (to indicate "not"), and edgy typography.

**Unicode source:** Combining Diacritical Marks block (U+0300–U+036F). All processing runs locally in your browser.

## Example

**Input:** `DELETE`  
**Output:** `D̸E̸L̸E̸T̸E̸`
