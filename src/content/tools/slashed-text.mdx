# Slashed Text Generator Online

Add diagonal slash marks through your text using Unicode combining characters. Perfect for crossed-out aesthetics, edits, and strikethrough alternatives. Free online.

## How Slashed Text Works

This tool uses the Unicode **Combining Long Solidus Overlay** (U+0338) and **Combining Short Solidus Overlay** (U+0337) characters. These are zero-width combining marks that draw a slash through the preceding character. Unlike standard strikethrough (which uses a horizontal line), slashed text uses diagonal marks for a more aggressive crossed-out look.

**Unicode source:** Combining Diacritical Marks block (U+0300–U+036F). All processing runs locally in your browser.
