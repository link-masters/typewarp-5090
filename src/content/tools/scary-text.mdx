# Scary Text Generator Online

Create terrifying scary text with dark Unicode styles and horror-themed effects. Perfect for Halloween bios, horror Discord servers, and spooky profiles. Free.

## How This Text Style Works

This tool applies combining diacritical marks and alternate Unicode characters (Fraktur, Runic, Cyrillic blocks) that visually resemble distorted or ominous versions of Latin letters. The effect is purely visual — the text remains standard, searchable Unicode.

All processing is client-side. Your text is never sent to any server.

## Example

**Input:** `Hello`  
**Output:** `H̷e̷l̷l̷o̷`

## Unicode Blocks Used

This tool uses **Combining Diacritical Marks** (Unicode range U+0300–U+036F). These are zero-width characters that stack above, below, or through a base character. By layering multiple combining marks per letter, the text appears distorted or "glitched." This technique is sometimes called "Zalgo text" after an internet meme.
