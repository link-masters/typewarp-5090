# Old English Text Generator

Create traditional Old English gothic text using Unicode blackletter characters. Perfect for historical bios, crest-themed profiles, and noble aesthetics. Free.

## How Gothic/Old English Text Works

This tool uses Mathematical Fraktur characters (U+1D504-U+1D537) from Unicode. Fraktur is a blackletter typeface historically used in German printing. The Unicode Mathematical Alphanumeric Symbols block preserves these letterforms for mathematical use.

The output uses standard Unicode and works on any modern device without installing fonts.
## Unicode Blocks Used

This tool uses characters from the **Mathematical Alphanumeric Symbols** block (Unicode range U+1D400–U+1D7FF). This block contains bold, italic, script, fraktur, monospace, and double-struck variants of Latin letters A–Z, a–z, and digits 0–9. These characters were originally standardized for use in mathematical notation (ISO/IEC 10646).
