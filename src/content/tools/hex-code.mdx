### The Science of Hex Decoding

Technically, your computer doesn't see "Letters." It sees numbers. "Hexadecimal" (Base-16) is a way of representing those numbers using 16 symbols ($0-9$ and $A-F$).

- **The Byte Map:** A character like `A` is mapped to the decimal value $65$, which in hexadecimal is `41`.
- **The Logic:** Our **hex converter** scans the UTF-8 byte array of your input. Each 8-bit byte is split into two 4-bit "nibbles," which are then converted into their corresponding hex glyphs.
