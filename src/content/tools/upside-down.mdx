## Frequently Asked Questions

<FAQ>
<FAQItem q="How does upside down text work?">

We map each letter to a visually similar Unicode character that looks like the inverted version. For example, 'a' becomes 'ɐ' and 'e' becomes 'ǝ'. The text is then reversed so it reads correctly when flipped.

</FAQItem>
<FAQItem q="Can everyone see upside down text?">

Yes! These are standard Unicode characters supported by all modern devices. Over 99% of users on iOS, Android, and desktop will see the flipped text correctly.

</FAQItem>
