# Subscript Generator Online

Generate subscript Unicode text for chemical formulas, mathematical notation, and unique social media bios. Works on all platforms. Free, instant online.

## How Cursive/Script Text Works

This tool uses Mathematical Script characters (U+1D49C-U+1D4CF) and Bold Script characters (U+1D4D0-U+1D503) from Unicode. These were originally encoded for mathematical notation but produce flowing, connected-looking text when used decoratively.

Processing is entirely client-side. Copy the output and paste it anywhere that supports Unicode.

## Example

**Input:** `H2O`  
**Output:** `H₂O`

## Unicode Blocks Used

This tool uses characters from the **Mathematical Alphanumeric Symbols** block (Unicode range U+1D400–U+1D7FF). This block contains bold, italic, script, fraktur, monospace, and double-struck variants of Latin letters A–Z, a–z, and digits 0–9. These characters were originally standardized for use in mathematical notation (ISO/IEC 10646).
